# Global Coastal Cleanup Data Visualization

This project analyzes coastal cleanup data from the International Coastal Cleanup (ICC) program
to identify trends in marine pollution across time and geography.

## Objectives
- Analyze coastal cleanup trends over time
- Identify the most common waste items collected
- Prepare clean, analysis-ready data for visualization and insights

## Data Source
The dataset comes from the International Coastal Cleanup (TIDES) platform by Ocean Conservancy.

## Project Structure
- data/raw  
  Original raw dataset collected from ICC

- data/processed  
  Cleaned and processed dataset used for analysis

- notebooks  
  Jupyter notebooks documenting data cleaning and analysis steps

## Notebooks
- 02_data_cleaning.ipynb  
  Data cleaning, preprocessing, and feature engineering

## Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook
- GitHub
- Tableau (for visualization)
